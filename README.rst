Nirdizati encoders
--------

Import and use SimpleIndexEncoder::

    >>> from nirdizati.encoders import SimpleIndexEncoder
    >>> simple_index_encoder = SimpleIndexEncoder()
    >>> encoded_trace = simple_index_encoder.encode_trace(data)

::
